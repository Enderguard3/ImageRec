Reconnaissance de plans et d'activités

Application de reconnaissance de plans et d'activités pour un robot assistant.

Développée pour un projet de recherche du Laboratoire de recherche en Gestion, Diffusion et Acquisition des Connaissances (GDAC-LIA) de l'Université du Québec à Montréal (UQAM). Le projet est porté par le réseau de recherche d'AGE-WELL (Aging Gracefully across Environments using Technology to Support Wellness, Engagement and Long Life) financé par les Réseaux de Centres d'Excellence du Canada (Gouvernement du Canada).

Dépendances

  • C++ 11
  • CMake 3
  • Python 3
  • Qt 5
  • OpenCV 3
  • Intel RealSense 2
  • Darknet

Les dépendances peuvent être installées en exécutant le script suivant (10 Go seront nécessaires pendant l'installation) :

sudo sh install.sh

Usage

Ce programme peut être compilé et testé par les commandes :

cd build
cmake ..
make
./ImageRec
